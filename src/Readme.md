# Папка проекта src на React

* `components ` - папка со всеми компонентами проекта
* `pages` - папка с страницами, которые используют компоненты для формирования

# Именования компонентов

`Header` - Шапка сайта
`Footer` - Подвал сайта
`CitiesBar` - Навигация по городам
`MainContent` - Основной контент главной страницы
`LocationsInfo` - Информация о локациях
`SearchResult` - Количество найденных локаций в выбранном городе
`SortingForm` - Форма сортировки результатов
`Sorting` - Варианты сортировки результатов
`Offers` - Предложения по локациям
`Map` - Карта
